## Core Insight: Universal Flow Patterns

### What UFIS Really Does (Abstracted)

1. **Input Ingestion**: Reads structured/unstructured information about "how things work"
2. **Flow Extraction**: Identifies steps, dependencies, resources, constraints
3. **Relationship Mapping**: Builds graph of what depends on what
4. **Optimization Detection**: Finds parallel opportunities, bottlenecks, inefficiencies
5. **Query Intelligence**: Answers questions about the system
6. **Simulation**: Models "what if" scenarios

**This pattern applies to ANY domain with processes!**

### Current UFIS (Software Only)

|-----------|----------------|---------|
| **Code** | Function calls, API routes, control flow | Call graphs |

**Example:**

- **Input**: Quality SOPs, inspection checklists, defect logs
